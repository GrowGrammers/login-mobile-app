# login Mobile App
React Native 기반 모바일 로그인 데모 앱

🚀 빠른 시작
1. Auth-Core 클론  
프로젝트 상위 디렉토리에 auth-core 클론
```
git clone https://github.com/your-org/auth-core.git ../auth-core
```
2. 디렉토리 구조
```
your-workspace/
├── auth-core/           # ← 여기에 클론
└── login-mobile-app/    # ← 현재 프로젝트
```
3. 설치
```
npm install
```

🔧 실행  
```
npm run android  # Android 실행
npm run ios      # iOS 실행
```
